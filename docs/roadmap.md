# Roadmap

## Fase 1 - MVP (0-3 meses)

- Upload PDF/XML/ZIP.
- Extraccion base + reglas contables iniciales.
- Clasificacion IA asistida.
- Exportacion ERP CSV/API basica.
- RBAC inicial y auditoria.

## Fase 2 - Produccion (3-6 meses)

- Multi-tenant robusto con RLS.
- CI/CD completo + observabilidad + runbooks.
- SLA operativos, backup y DR.
- Integraciones ERP prioritarias Colombia.

## Fase 3 - Escalabilidad (6-12 meses)

- Autoscaling avanzado y particionado de jobs.
- Event bus para analitica near real-time.
- Feature flags, canary y blue/green.
- Bodega de datos y BI.

## Fase 4 - Enterprise (12+ meses)

- Multi-region y data residency.
- SSO enterprise (SAML/OIDC corporativo).
- Contract testing y chaos engineering.
- Service mesh si se separan microservicios.
